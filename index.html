1. Scheduling von Tasks

Problemstellung:

Gegeben ist eine Liste von Tasks, die jeweils eine Dauer und eine Deadline haben. Ziel ist es, möglichst viele Tasks vor ihrer Deadline zu erledigen.

  • Task 1: Dauer 3, Deadline 6
  • Task 2: Dauer 2, Deadline 8
  • Task 3: Dauer 1, Deadline 4
  • Task 4: Dauer 4, Deadline 7
  • Task 5: Dauer 2, Deadline 5
